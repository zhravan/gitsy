### Prerequisites

- Node 18+
- npm
- Go 1.21+
- Wails CLI (`go install github.com/wailsapp/wails/v2/cmd/wails@latest`)

### Install

```bash
make install
```

### Develop

```bash
make dev
```

### Build

```bash
make build
```
